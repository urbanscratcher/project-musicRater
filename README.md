# MusicRater

## 목적

- 음악 검색 및 관리
- useState, useEffect, useRef, custom use hook 연습
- React, React-DOM만 사용

## 기능

- 유튜브에서 음악 검색
- 추천 검색어 제공
- 평점 남기기
- 평점, 영상 길이 평균
- 남긴 평점 목록 + 필터링
- 플레이리스트 평점, 영상 길이 평균
- 플레이리스트 담기, 삭제
- 플레이리스트 순차+랜덤 재생

## 사용 라이브러리 & 기술

- Vite + React w/ pnpm
- TailwindCSS
- Docker
- ytmusicapi
- react-youtube (YouTube IFrame API)
- react-top-loading-bar
